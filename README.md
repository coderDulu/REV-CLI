# React + Electron + Vite

## Use

```sh
# install
1. npm install

# dev
2. npm run dev

# build
3. npm run build

```

## 相关技术

| 描述       |       相关       |
| :--------- | :--------------: |
| 框架       |     Electron     |
| UI 框架    |      React       |
| 打包工具   |       Vite       |
| 语言       |    TypeScript    |
| 主进程编译 |      rollup      |
| 打包       | electron-builder |
| css        |   TailwindCss    |
